Ichor: Infrared Camera Accessor

Access and record infrared cameras on Windows devices

Download camera_tool.exe

Overview

This tool allows users to access and record the infrared camera on Windows devices (typically used for facial recognition logins).

Running the Program

Run in Command Prompt:

camera_tool

Or run in MSYS2 MinGW64 terminal:

./camera_tool

Make sure you are in the same directory as the executable.

Main Program Controller - main.c

main.c
// Main program flow
int main(void) {
    WCHAR instanceId[1024];

    // Part 1: Let user select a camera
    int ret = SelectCameraInstanceId(instanceId, sizeof(instanceId) / sizeof(WCHAR));

    // Part 2: Query supported camera formats
    ret = PrintCameraFormatsForInstanceId(instanceId);

    // Part 3: Live preview and recording
    ret = PreviewIrCamera(instanceId, recordSeconds, outputFile);
}

Compilation Instructions

To compile this program in your MSYS2 MinGW64 environment do:

gcc main.c camera_select.c camera_formats.c ir_capture.c -o camera_tool.exe \
    -lsetupapi -lcfgmgr32 \
    -lmf -lmfplat -lmfreadwrite -lmfuuid -lole32 -luuid \
    -luser32 -lgdi32 -lavifil32

Read documentation.txt if you want more information on the code.

Compatibility Note

This program currently only works on certain Windows devices with an infrared camera installed. If you have a black dot (IR sensor) next to your camera, this program may work on your device. Most devices with facial recognition login have this sensor.
