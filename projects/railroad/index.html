Railroad

An Encrypted-Group-Task Manager

ABOUT

A simple task system manager, in order to track what is being done and is left to be done. Encrypted to allow people to publicly share keys to friends. Only those that know the password can access the encrypted tasks. Built for figuring out what's going on and privacy.

FEATURES

Use F2 on login page to demo
Can assign task + subtask to people
Can override previous assignments
Visual merge diff preview
Use F3 on merge page to test merge
Works offline
